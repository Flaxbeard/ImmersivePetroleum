Штанговый насос
Давай-давай, выкачивай!
<&pumpjack0>Штанговый насос - многоблочная структура, добывающая нефть из месторождений под коренной породой. Формируется ударом молота по тяжёлому инженерному блоку во втором слое.<np>
Штанговый насос использует <config;i;pumpjack_consumption> Flux/t для добычи <config;i;pumpjack_speed> mB/t нефти. Полученная нефть автоматически выталкивается с боковых портов, помеченных оранжевыми точками.

Средних размеров месторождение будет исчерпано примерно за <config;i;pumpjack_days> дней. Добавление большего количества штанговых насосов ускорит этот процесс.

Подача редстоун-сигнала к панели управления будет останавливать работу машины. Поведение панели управления можно инвертировать молотом инженера.
